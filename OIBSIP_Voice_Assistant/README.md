# 🎤 Voice Assistant - OIBSIP

## 📌 Description
This is a Python-based Voice Assistant that performs tasks using voice commands.

## 🚀 Features
- Speech Recognition
- Text-to-Speech
- Tell time and date
- Wikipedia search
- Open websites

## 🛠️ Technologies Used
- Python
- SpeechRecognition
- pyttsx3
Wikipedia

## ▶️ How to Run
1. Install dependencies
2. Run: python assistant.py

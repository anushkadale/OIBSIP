# 🧮 BMI Calculator - OIBSIP Internship Project

## 📌 Description
This is a Python GUI-based BMI Calculator developed using Tkinter. It calculates BMI based on user height and weight and displays the BMI category.

## 🚀 Features
- User-friendly interface
- BMI calculation
- Health category detection
- Input validation
- Clear button

## 🛠️ Technologies Used
- Python
- Tkinter

## ▶️ How to Run

Run:

python bmi_calculator.py
